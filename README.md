# microfluidic-simulation-ai-optuna
# 微流控仿真与AI参数优化工程作品集
基于 PyFluent + Optuna 实现微流控器件仿真自动化与参数反演设计。

## 项目亮点
- 实现仿真流程自动化，减少手动操作 80%
- 使用 TPE 算法优化微通道结构，提升混合效率/聚焦稳定性
- 支持桌面端运行，降低工程落地成本
- 提供完整复现流程与工程报告

## 技术栈
- PyFluent / Fluent
- Python, NumPy, Pandas, Matplotlib
- Optuna (参数优化)
- Gmsh / ANSYS Meshing
